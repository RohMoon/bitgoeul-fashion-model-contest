📧 모델 콘테스트 이메일 폼 설정 가이드

💡 안내: 이 페이지는 참가 신청 폼을 이메일로 받기 위한 설정 가이드입니다. 파일 첨부를 포함한 완전한 이메일 폼 기능을 제공합니다.

📌 이메일 전송 방법 선택

옵션 1: EmailJS (추천)

특징: 무료 티어 제공, 간단한 설정

✅ 장점: 서버 불필요, 파일 첨부 가능, 빠른 설정
❌ 단점: 월 200건 제한 (무료)

옵션 2: Formspree

특징: 폼 전용 서비스

✅ 장점: 매우 간단, 스팸 필터링
❌ 단점: 월 50건 제한, 파일 첨부 유료

옵션 3: Google Forms

특징: Google 서비스 연동

✅ 장점: 완전 무료, 무제한
❌ 단점: 커스터마이징 제한

🚀 EmailJS 설정 방법 (추천)

1 EmailJS 가입

1. EmailJS.com 접속

2. 무료 계정 생성 (Sign Up Free)

2 이메일 서비스 연결

1. Dashboard → Email Services → Add New Service

2. Gmail 선택 → Connect Account

3. Google 계정으로 인증

3 이메일 템플릿 생성

1. Email Templates → Create New Template

2. 아래 템플릿 내용 복사:

Subject: [모델 콘테스트 신청] {{name}} - {{phone}} 새로운 모델 콘테스트 참가 신청이 접수되었습니다. ━━━━━━━━━━━━━━━━━━━━━━━━ 📋 신청자 정보 ━━━━━━━━━━━━━━━━━━━━━━━━ 이름: {{name}} 생년월일: {{birth}} 연락처: {{phone}} 이메일: {{email}} 신장: {{height}}cm 경력: {{experience}} 포트폴리오: {{portfolio}} ━━━━━━━━━━━━━━━━━━━━━━━━ 💬 지원동기 ━━━━━━━━━━━━━━━━━━━━━━━━ {{motivation}} ━━━━━━━━━━━━━━━━━━━━━━━━ 📎 첨부파일 ━━━━━━━━━━━━━━━━━━━━━━━━ {{attachment}} ━━━━━━━━━━━━━━━━━━━━━━━━ 신청 일시: {{date}} IP 주소: {{ip_address}}
4 API 키 복사

1. Account → API Keys에서 Public Key 복사

2. 아래 코드의 YOUR_PUBLIC_KEY 부분에 붙여넣기

💻 구현 코드

index.html 파일의 </body> 태그 직전에 추가:

<!-- EmailJS SDK --> <script type="text/javascript" src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script> <script type="text/javascript"> // EmailJS 초기화 (function() { emailjs.init("YOUR_PUBLIC_KEY"); // 여기에 Public Key 입력 })(); // 폼 제출 처리 document.getElementById('applicationForm').addEventListener('submit', function(e) { e.preventDefault(); // 제출 버튼 비활성화 const submitBtn = this.querySelector('.submit-btn'); const originalText = submitBtn.textContent; submitBtn.textContent = '전송 중...'; submitBtn.disabled = true; // 폼 데이터 수집 const formData = { name: document.getElementById('name').value, birth: document.getElementById('birth').value, phone: document.getElementById('phone').value, email: document.getElementById('email').value, height: document.getElementById('height').value, experience: document.getElementById('experience').value, portfolio: document.getElementById('portfolio').value || '없음', motivation: document.getElementById('motivation').value || '없음', date: new Date().toLocaleString('ko-KR'), ip_address: 'Web Form' }; // 파일 첨부 처리 (선택사항) const fileInput = document.getElementById('portfolioFile'); if (fileInput && fileInput.files[0]) { const file = fileInput.files[0]; formData.attachment = `파일명: ${file.name} (크기: ${(file.size/1024).toFixed(2)}KB)`; } else { formData.attachment = '첨부파일 없음'; } // EmailJS로 전송 emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', formData) .then(function(response) { alert('신청이 완료되었습니다! 빠른 시일 내에 연락드리겠습니다.'); document.getElementById('applicationForm').reset(); }) .catch(function(error) { alert('전송 중 오류가 발생했습니다. 다시 시도해주세요.'); console.error('EmailJS Error:', error); }) .finally(function() { submitBtn.textContent = originalText; submitBtn.disabled = false; }); }); </script>

📎 파일 첨부 기능 추가 (선택사항)

폼에 파일 업로드 필드 추가:

<div class="form-group"> <label for="portfolioFile">포트폴리오 파일 (PDF, JPG, PNG - 최대 5MB)</label> <input type="file" id="portfolioFile" name="portfolioFile" accept=".pdf,.jpg,.jpeg,.png" onchange="validateFile(this)"> <small style="color: #666;">프로필 사진이나 포트폴리오를 첨부해주세요</small> </div> <script> function validateFile(input) { const file = input.files[0]; const maxSize = 5 * 1024 * 1024; // 5MB if (file && file.size > maxSize) { alert('파일 크기는 5MB를 초과할 수 없습니다.'); input.value = ''; return false; } const allowedTypes = ['application/pdf', 'image/jpeg', 'image/png']; if (file && !allowedTypes.includes(file.type)) { alert('PDF, JPG, PNG 파일만 업로드 가능합니다.'); input.value = ''; return false; } return true; } </script>

🔄 대체 방법

방법 1: Formspree (간단)
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST" enctype="multipart/form-data"> <input type="text" name="name" required> <input type="email" name="email" required> <input type="file" name="upload"> <button type="submit">전송</button> </form>
방법 2: Google Forms 임베드

1. Google Forms에서 폼 생성

2. 보내기 → 임베드 HTML 복사

3. 웹사이트에 iframe으로 삽입

방법 3: Netlify Forms (Netlify 호스팅 시)
<form name="contact" method="POST" data-netlify="true" data-netlify-honeypot="bot-field"> <input type="hidden" name="form-name" value="contact"> <input type="text" name="name"> <input type="email" name="email"> <button type="submit">전송</button> </form>
